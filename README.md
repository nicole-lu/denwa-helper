# DenwaHelper 電話ヘルパー

AI assistant for Japanese phone calls. Built with Gemini 3.5 Flash.

## Features
- Real-time Japanese phone call translation
- IVR menu parsing — tells you which button to press
- Suggested Japanese replies
- Bilingual output (English + Chinese)

## Tech Stack
- Gemini 3.5 Flash (Google Cloud AI)
- Streamlit
- Python / PyAudio
